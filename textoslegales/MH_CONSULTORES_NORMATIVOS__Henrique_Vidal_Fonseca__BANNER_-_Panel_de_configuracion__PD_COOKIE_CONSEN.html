PANEL DE CONSENTIMIENTO DE COOKIES PD COOKIE CONSENT
1. Funcionalidad que inserta un panel de consentimiento de cookies y bloquea los elementos que
pudieran insertarlas
PD Cookie Consent
es una librera JavaScript a travs de la cual se pretende facilitar el bloqueo de aquellos
elementos del DOM que pudieran incrustar cookies y sirve para que el tcnico del sitio web pueda insertar una
ventana emergente (banner) que avise al visitante sobre la existencia de cookies, adems de facilitar un medio
para la gestin de aquellos elementos que pudieran generarlas, bloquendolos y evitando que las generen
hasta el posterior desbloqueo por parte del visitante.
2. Cmo insertar la librera en nuestro sitio web?
MUY IMPORTANTE
PASO 1: insertar la siguiente librera y sus estilos entre las etiquetas quedando el script de la librera por
encima de cualquier otro que pudiera haber entre dichas etiquetas:
<head></head>
Si no se cumple este paso tal como se describe, PD Cookie Consent no funcionar correctamente.
<link rel="stylesheet" href="https://pdcc.gdpr.es/pdcc.min.css">
<script charset="utf-8" src="https://pdcc.gdpr.es/pdcc.min.js"></script>
<script type="text/javascript">
PDCookieConsent.cong({
"brand": {
"dev" : true,
"name": "MH Consultores Normativos",
"url" : "https://mhconsultoresnormativos.com",
"websiteOwner" : ""
},
"cookiePolicyLink": "",
"hideModalIn": [""],
"styles": {
"primaryButton": {
"bgColor" : "#A1FFA1",
"txtColor": "#036900"
},
"secondaryButton": {
"bgColor" : "#EEEEEE",
"txtColor": "#333333"
}
}
});
</script>

3. Cmo funciona PD Cookie Consent?
Cuando un visitante accede al sitio web, la librera
PD Cookie Consent
observa cada uno de los elementos
existentes en la pgina visitada y, si alguno de estos cumple los requisitos necesarios para ser bloqueado, se
bloquea.
Una vez observados todos los elementos de la pgina, la librera mostrar el panel
PD Cookie Consent
y
generar una cookie de sesin nombrada pd_cc, necesaria para el correcto funcionamiento de la pgina.

4. Mensajes informativos sobre los elementos bloqueados
Adems del panel PD Cookie Consent, la librera tambin proporciona un mensaje que aparecer en la
parte superior del sitio web e informar al visitante de los elementos visuales que hayan sido bloqueados
y que, en consecuencia, no se estn visualizando.
Ante este mensaje el visitante podr:
Informarse sobre el contenido bloqueado y que no se est visualizando, incluyendo la lista de los
elementos bloqueados (en el caso de la imagen proporcionada, YouTube).
En relacin con la lista de elementos bloqueados podr efectuar las siguientes acciones:
Aceptar todos los elementos bloqueados de la pgina visitada haciendo clic en Aceptar
todas, quedando estos activos en el resto del sitio web (si existiesen). Una vez pulsado el
botn se ocultar el mensaje.

cualesquiera de las acciones ya mencionadas en este mismo documento.
